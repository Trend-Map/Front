# 트렌드 맵 — Design Context

## Design Context

### Users
20대 한국인 트렌드 민감 소비자. 지금 어디서 무엇이 뜨는지를 빠르게 파악하고 싶다. 첫 진입은 음식/카페 트렌드이며, 이후 패션·라이프스타일로 확장 예정. 이동 중에 모바일로 접속하는 경우가 많고, 정보를 빠르게 스캔한 후 즉각 행동(방문, 제보, 공유)으로 전환하는 사용 패턴을 가진다.

### Brand Personality
**3 words:** 빠른 · 직관적인 · MZ한

지금 일어나고 있는 일을 실시간으로 반영하는 느낌. 과도한 설명 없이 숫자와 지도와 핀으로 말한다. 트렌드를 발견했을 때 오는 가벼운 흥분감을 준다. 장황하지 않고, 친절하되 가볍지 않다.

**Voice/Tone:** 중간 어조 — 반말도 존댓말도 아닌 짧은 명령형/청유형. "지금 확인하기", "제보하기", "바로 보기". 문장은 짧게 끊는다.

### Aesthetic Direction
- **Visual tone:** 편집 디자인과 데이터 인터페이스의 중간. 여백은 있되 비어 보이지 않는다. 카드 간 구분은 섀도가 아닌 구조와 여백으로.
- **색상 팔레트:**
  - Accent: 버밀리언/앰버 계열 (예: `#E84C2B` / `#F59E0B`)
  - Text: Deep charcoal (`#1A1A1A`)
  - Background: Off-white (`#F8F7F5`)
  - Pin — available: `#22C55E` (green) / soldout: `#EF4444` (red) / unknown: `#94A3B8`
- **Typography:** Noto Sans KR. Weight contrast를 강하게 — 제목 700–900, 본문 400, 보조 정보 300. 폰트 크기 계단은 12/14/16/20/28/40px 기준.
- **References:** dubaicookiemap.com, buttertteokmap.com — 구조만 참고, 시각적 개성은 초과할 것
- **Anti-reference:** 파스텔 컬러, 둥근 카드+그림자+그라디언트가 난무하는 한국 양산형 앱 스타일
- **Mode:** Light only

### Design Principles

1. **지도가 제품이다** — 지도는 장식이 아닌 주 인터페이스. 어느 화면에서든 지도로 돌아가는 경로가 한 탭 이내여야 한다.

2. **신호는 즉시 읽혀야 한다** — 핀 색상(green/red), 재고 레벨, 트렌드 강도는 텍스트 없이도 상태를 전달해야 한다. 색상과 아이콘이 1차, 텍스트는 2차.

3. **살아있는 느낌** — 타임스탬프, 언급 수, 최근 제보 시각이 항상 보여야 한다. 정적인 리스트가 아니라 지금 이 순간의 스냅샷임을 느끼게.

4. **장식 없이 성격을 만든다** — 개성은 굵기 대비·자간·여백 리듬으로 만든다. 그림자, 그라디언트, 과한 라운드는 쓰지 않는다.

5. **확장을 전제한 중립성** — 음식에 특화된 아이콘·언어를 피하고, 패션·굿즈·이벤트로 확장해도 시스템이 깨지지 않는 구조를 유지한다.
